# Passat

<p class="passat-tagline">IntelliJ プラットフォーム上に構築された、オープンプラットフォームの Object Pascal IDE。</p>

Passat は IntelliJ プラットフォームに第一級の **Object Pascal** サポートをもたらします。**無償の
Object Pascal コンパイラ**の上に構築され、**Free Pascal コンパイラ（FPC）**をリファレンスとなる
ツールチェーンとして採用しています。コンパイラと言語バージョンの扱いは抽象化されているため、
将来的に他の無償 Object Pascal コンパイラを追加できます。

Passat は**プラグインファースト**で開発されています。既存の JetBrains IDE にインストールできる
プラグインとして提供され、後の段階では同一のコードベースから**スタンドアロンの IDE** 配布物も
パッケージ化されます。

## 機能

- **専用の Pascal プロジェクト／モジュールタイプ** —— Java や Kotlin のモジュールと同様に、各 Pascal
  モジュールはターゲットコンパイラ（SDK として登録された FPC インストール）、ターゲット言語バージョン
  （Object Pascal の方言レベル）、独自の依存関係リストを持ちます。
- **完全な言語サポート** —— PSI ツリーを生成するレキサーとパーサー、シンタックスハイライト、コード補完、
  ナビゲーション（宣言へ移動／使用箇所の検索）、構造ビュー、リファクタリング、インスペクション。
- **ビルドと実行** —— 専用の実行構成を通じて、設定したコンパイラでプロジェクトをコンパイル・実行します。
- **デバッグ** —— ブレークポイント、ステップ実行、変数の確認、式の評価といった完全な IDE デバッグ機能。
- **デュアルデリバリー** —— 一つのコードベースから二つの製品を提供：既存の JetBrains IDE 向けの
  インストール可能なプラグイン**および**スタンドアロンの Passat IDE。

## ロードマップ

Passat は段階的に構築されます：

1. **言語パーサーと PSI** —— レキサー、文法、PSI、基本的なハイライト。
2. **プロジェクト／モジュールモデル** —— コンパイラ SDK、言語バージョン、依存関係を持つ Pascal モジュールタイプ。
3. **ビルドと実行** —— FPC によるコンパイルと実行、実行構成。
4. **デバッグ** —— 完全なデバッグ機能セット。
5. **スタンドアロン IDE のパッケージング** —— プラグインに加えて、Passat を独自の IDE として配布。
